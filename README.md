# Band Finder Project

## Description
This is a web application designed to help people find music bands and events. It also allows band owners to manage their profiles, public concerts, and private booking requests.

## Technology Stack
* Frontend: HTML, CSS (Bootstrap), and JavaScript (jQuery).
* Backend: Node.js with the Express framework.
* Database: MySQL for storing users, bands, events, and messages.
* AI Integration: Google Gemini AI for the chatbot and smart filtering.
* Maps: OpenStreetMap (OpenLayers).
* Weather: OpenWeatherMap API.
